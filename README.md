# OutfitSwap
A BepInEx Gorilla Tag mod that switches to your next saved outfit when you double tap the A button on your right controller

## Installation
1. Download the latest `OutfitSwap.dll` from the [Releases page](https://github.com/struuct/OutfitSwap/releases)
2. Place the `OutfitSwap.dll` file in your `BepInEx/plugins` folder
3. Launch the game, when you join a room, all players should have the minecraft nametags above them
